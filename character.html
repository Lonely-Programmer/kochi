人物介绍篇 | Characters

前言

小说里涉及到的人物还不少，你是不是已经记不清名字了呢？没关系，在这里我们会对所有的人物进行介绍，如果忘了的话推荐到这里找一找。

为了方便查阅，我们按照身份对角色进行分组整理。当然，有的角色不止一个身份，这种情况下以主要身份为准。

这部分内容非常长，内容也比较分散，所以不建议一次全都看完。可以边看小说边结合着看。

第一部分：古津结界内侧

在古津结界的内侧生活着的人们……

1.1：历史人物

造就了今天的人们

初明 伊吕波

初明 伊吕波 / 初はつ明あき　いろは / Hatsuaki Iroha

一个具有传奇色彩的人物，出生于闰年的2月29日，也是大名鼎鼎的古津结界的建立者。

关于结界建成的故事，如果感兴趣的话可以去看一下《石阶上的神秘数字》或者《古津灵异实录》。当然如果懒得看的话随便找一个人问一问也就有了答案，因为实在太有名了。

这里简单介绍一下。她在当年对灵力有着十分深入的研究，提出了很多理论。有一天，她发现灵力常数正在下降，如果任其发展下去会给人们带来灾难，而阻止的唯一方法就是建造一个结界。

但是建造结界有很大的危险性，而且难度极大。她为了不让别人陷入危险，一个人建造结界。最终，结界建造完成了，这也就是今天的古津结界。但她也因体力不支而英年早逝了。据说她的灵魂一直存在于古津结界的内部……

和结界同时流传下来的，还有一个非常有名的场景，也就是本篇的封面图。一个年轻的女孩目光坚定地用着自己最后的一点力气，建造着结界，在树林里发出一丝丝光芒。结界建造的最后一天，她和朋友们嘱托完后事以后，躺在湖边的石阶上。天空中美丽的繁星，周围飞舞着的萤火虫，倒映在湖面上。看着这些，她微笑着闭上了眼睛。随后，身体化作萤火虫一般发光的微粒，随风飘向远方……

五十岚 茜

五十岚 茜 / 五十嵐いがらし　茜あかね / Igarashi Akane

她和伊吕波曾经一起进行灵力方面的研究，掌握了很多这方面的知识，因此被伊吕波指定为结界维护者之一。

她在结界维护的早期进行了大量的研究和观测工作，并将此运用于实践中。为了方便使用，她把一些最常用的数据刻在了结界附近的湖边的石阶上，这也就是石阶上神秘数字的由来。

她也是神秘社团的创始人之一，第一任会长，和尔穗一起成立了神秘社团，以得到更多有志之人的帮助。早期的结界还很不稳定，社团的人数也只有四五人，所以她花费了大量的心血。在她的努力之下，结界逐渐趋于稳定。

似乎是为了给神秘社团营造仪式感，她去东风神社给每个成员求取了护身符，作为成员的凭证。不过后来发现护身符所蕴含的能量对维护结界确实是有帮助的，因此这个习俗至今还有。

当然，也有一些已经消失的习俗。例如当年还有着成员之间要用敬语交流的规定，但现在已经基本取消了。

尽管她掌管的是关系到整个世界命脉的结界，但她却始终坚持不主动宣传社团，因为她担心信仰过于集中所带来的问题。在自己老去的时候，她也决定隐居起来，不再参与结界的维护工作，这样做相当于是放弃自己所收集的信仰。

可能正是因为前人这样的自觉，才让人不会因为贪求信仰而被欲望迷惑，让结界外侧的悲剧重演。

初明 尔穗

初明 尔穗 / 初はつ明あき　にほ / Hatsuaki Niho

她是伊吕波的妹妹，不过因为学业的缘故对灵力的研究不多。她也是神秘社团的创始人之一，第一任副会长。

当年的神秘社团，会长的职权确实比副会长大。因为她一开始不太懂怎么维护结界，所以把会长的重任交给了茜。但因为结界的一些高级操作，例如穿越结界，需要和结界的建造者体质相近才能安全完成，所以这个任务就交给了她。

完全不会的她通过长期的摸索，在茜的帮助下，终于找到了安全打开结界的方法，同时也发明了把此方法用物理方式转移给其他人的技术。

随着时间的流逝，她维护结界的能力也和茜差不多了，于是她在卸任前把结界维护的技术传授给女儿初明千梨，让她继任副会长这一有体质要求的独特职位。从此，副会长的职权超过了会长。

和茜一样，她在卸任以后就隐居起来。对她来说，能够实现维护结界这一理想，远比获得名利要重要得多。

初明 千梨

初明 千梨 / 初はつ明あき　千ち梨り / Hatsuaki Chiri

神秘社团的第2任副会长。她把前辈的结界维护方法进行了系统的总结，并进行了一定的优化和创新，形成了现在结界维护的雏形。

初明 霞

初明 霞 / 初はつ明あき　霞かすみ / Hatsuaki Kasumi

神秘社团的第3任副会长，羽衣的外婆。从她开始，神秘社团也出现了一些变化，例如副会长在女儿18岁的时候卸任，社团内部也慢慢地不再用敬语称呼。

初明 茉莉子

初明 茉莉子 / 初はつ明あき　茉ま莉り子こ / Hatsuaki Mariko

神秘结界上一任副会长，羽衣的妈妈。根据神秘社团的传统，她在羽衣18岁那年让羽衣担任副会长的职位，自己则到一个叫伊西的地方居住，意思是成为过去。（日语“伊西へ”和“古”发音相同）

听说她偶尔会到古津大学，偷偷地看看女儿的情况。

藤本 月季

藤本 月季 / 藤ふじ本もと　月つ季き / Fujimoto Tsuki

神秘结界上一任会长。从古津大学毕业以后，她把会长的职位交给了芽吹。之后便过上了普通人的生活。

跟茉莉子的关系很好，私下经常用“Jasmine”（ジャスミン）称呼对方。

1.2：神秘社团

神秘社团的成员

初明 羽衣

初明 羽衣 / 初はつ明あき　羽は衣ごろも / Hatsuaki Hagaromo

性格十分开朗活泼的一个人，很喜欢和别人打交道。作为神秘社团的副会长，她具有最高决定权，但本人似乎完全没有那种威严，不知道的人完全看不出她的身份。

她平时保持着良好的作息时间，每天早上7点半准时起床。由于经常会没事干，所以她每天的生活经常是在散步、蹭课、发呆等等。据说她最喜欢去的社团是日语社。

怎么看她都像是个不务正业的人。不过尽管如此，她对结界的事情似乎还是十分重视的，只要有这方面的需求，无论在什么地方都会马上赶过去。特别是在需要穿过结界的时候，她会花费大量的时间进行准备。

据说她有着读心的能力，但是除了维护结界以外基本不会使用，所以很多人也并不清楚。另外，她还有着超乎常人的抗冻体质，就算是在下雪的时候穿夏天的衣服也只是感觉有点凉而已，而且以前还经常这样做。不过，在芽吹的再三说教之下，她现在已经正常多了。

羽衣从小就被妈妈当作下一任的副会长来培养，所以家教也比较严格。从大概6、7岁开始，她每天就要很早起床，跟着妈妈学习结界的各种原理，比如怎么维护结界、怎么去神社里求护身符、怎么跟会长合作等等，并且还要不断地使用自己的特殊能力以提高灵力值。为了锻炼吃苦耐劳的精神，冬天必须穿得很单薄。一般人在童年愉快地玩耍时，她一直在磨练着自己。

小时候艰苦的训练并没有压倒她。相反，她平时经常和社团的成员交流，从有趣的经历中找到乐子。

18岁那年，妈妈把副会长的重任交给了她，并按照传统隐居到了伊西。虽然一开始遇到了很多困难，但她最后还是学会了穿过结界的方法，成为了可以独当一面的结界管理者。

看似轻松的微笑，背后是常人难以想象的努力。对于每一个热爱着结界的人来说，结界的未来交给这样一个人，应该是很幸运的一件事情吧。

小鸟游 芽吹

小鸟游 芽吹 / 小鳥遊たかなし　芽め吹ぶき / Takanashi Mebuki

首先提醒一下，请不要把她的姓氏读错，否则会让人感觉没有文化……

言归正传。她是神秘社团的会长。听这个职位感觉像是神秘社团的最高负责人，但由于一些历史原因，会长实际上的职权是没有副会长大的。

结界大部分的日常事物都是由她负责的，这背后的原因可能是羽衣经常不务正业吧。她是一个能力很强的人，在兼顾学习的情况下还能把结界的日常工作漂亮地完成。

在性格方面，她十分沉稳，在言行上也有点不苟言笑，在社团的会议上，她也经常用敬语和羽衣对话。总觉得她是个内心有点坚硬的人，比羽衣看上去更像是神秘社团的负责人。

不过，不熟悉她的人不知道的是，她的内心其实也有柔软的部分。

她继任会长的时间点，恰好是羽衣继任副会长之后不就。那时的两人基本没有经验，只是凭借前人的教导探索维护结界的方法。那段时间，她们度过了十分艰难的日子，但也结成了非常牢固的友谊。

虽然在会议上两人会用正式的称呼，但平时她们都是直接叫对方名字的，而且还经常互相吐槽。关于两人的关系，感兴趣的话可以去看《古津境守物语》体会一下。

还是很羡慕这样的友谊的。

楠濑 七凪

楠濑 七凪 / 楠くす瀬のせ　七凪ななぎ / Kusunose Nanagi

据说她在刚进大学的时候有点不太合群，平时独来独往，跟别人的交集比较少，和室友们的对话也不多。

后来，她在社团招募活动时发现了神秘社团的存在，并加入其中。她很快就和羽衣成为了好朋友，并经常去羽衣那里串寝。

她的体质有些特别，不知道为什么可以从石楠花中获取能量，所以在四五月份的时候经常把石楠花带进寝室里，而这也导致了室友们的不满。

在室友们的压力下，她不得不把寝室里的石楠花全部处理掉。但与此同时，她也萌生了一个更为大胆的想法：如果能让校园里的石楠花一直开放下去，那么自己就可以源源不断地获取能力，不用再把花带进寝室了。

于是，她每天晚上给宿舍附近的几棵石楠树注入能量。如她所愿，石楠花直到六月底都没有凋谢。但另一方面，学校里一些过敏体质的学生已经开始受不了了，纷纷进了校医院。

出于某种原因，神秘社团并没有阻止她。阻止她的，是秋代、瞳、木灵三人。她被三人找到并击败，答应不再给石楠树注入能量。

三人的态度感染了她。自己明明用尽全力攻击，她们最后却没有责怪自己，还把自己搀扶回宿舍。她便和三人成为了朋友。

似乎正是从这件事以后，她开始更加注意别人的感受，也慢慢交到了更多的朋友。

天坂 久方

天坂 久方 / 天あま坂さか　久ひさ方かた / Amasaka Hisakata

神秘社团和古津大学为数不多的男生之一。名字的由来是出生于一个月圆之夜（日语中“久方”是“月”的枕词）。

后来和藤原美奈子谈了恋爱。基本还是比较普通的人吧，所以基本介绍完了。

三千代 时雨

三千代 时雨 / 三さん千ぜん代だい　時雨しぐれ / Sanzendai Shigure

神秘社团中非常特殊的一名成员。她本身不具备特殊能力，但对灵力的研究非常深入，发明了产生灵力的装置，成为了神秘社团的技术顾问

她来自古津结界的外侧。据说她的妈妈也是从事灵力研究的，找到了穿过古津结界的方法，但因为不希望自己的成果被统治者用来侵略而被逮捕。在此之前，她用自己发明的装置把时雨送到古津结界的内侧，然后破坏掉了装置。临行前，她告诉时雨要去寻找结界的维护者，并给予帮助。

她来到古津结界内侧的时候，只有12岁，被东风神社上一任巫女收养，除此以外几乎没有人知道她的存在（包括神秘社团）。

长大以后，她靠打工养活自己，同时在闲暇时间凭着自己的记忆做出了一些可以产生灵力的装置。当然，她没有忘记妈妈对自己的嘱咐。于是，她利用自己制作的机器收集信仰，希望用这种方式引起结界维护者的注意。

她成功了。神秘社团的成员羽衣、芽吹、七凪找到了她。在她说明了事情的来龙去脉以后，神秘社团决定把她作为秘密的顾问和技术支持。

听说她在社团里享受着特别的权利，比如不用参加会议，不用参加待宵的结界维护。会长和副会长会主动上门告诉她最新情况。

有了她带来的技术革新，神秘社团会应该会进入新的阶段吧。

冰上 一梦

冰上 一梦 / 氷上ひかみ　一いち夢む / Hikami Ichimu

严格来说她并不是神秘社团的成员，但她似乎和神秘社团的联系非常紧密，所以就放在了这里。

她是一位比较神秘的巫女，关于她的介绍和记载非常少，只知道她很小就开始在东风神社任巫女一职。由于本身去神社的人就不是很多，而且她在做完准备工作以后就很少露面，所以没有几个人见过她的真容。

平时神社的各种事物均由她来负责，而神秘社团的护身符又是在神社里求取的，所以按理来说她和社团的副会长应该是彼此认识的，但历届的副会长都很少提到巫女，《古津灵异实录》里也鲜有提及，因此线索还是不多。

羽衣曾经偶然提到过，自己小时候跟妈妈去神社的时候就见过她，她当年也就15岁，但却表现出不符合她年龄的沉稳和成熟，跟自己形成很大的反差，所以经常被妈妈用来做对比。这可能是关于她本人唯一能知道的事情了。

此外，关于她和结界与神秘社团的关系究竟是什么，以及她的身世，供奉的神是谁，这个神社和结界另一边相同位置的神社有什么关系，这些问题都充满了谜团，或许这些问题的答案永远都无法揭晓。

1.3：代码使

代码使们

伊仓 美香

伊仓 美香 / 伊い倉くら　みか / Ikura Mika

无论是第一次见到她，还是已经跟她比较熟悉，美香都给人一种优雅而沉稳的感觉。她是一名优秀的代码使，但在别人遇到困难的时候，她常常会倾囊相助，毫不吝啬。

她从小就有一定的天赋，学习成绩一直很好。但是，她并没有满足于此。在高中的时候，她接触到了编程，并产生了浓厚的兴趣。每天，她都会花几个小时在编程上。虽然当时的编程水平还比较稚嫩，但她还是十分享受编程的过程。

不过在高二后半段，升学压力渐渐增大，课业负担也越来越繁重，每天在一门高考不考的科目上花这么多时间显然是不现实的事情。于是，在父母的催促下，她暂时放下了自己最喜欢的编程。也正是出于对编程的热爱，她报考了古津大学的计算机专业。

在高考后的那个暑假，她偶然知道了记忆编辑技术，并在古津大学听了一个相关的讲座。由于当时正值暑假，所以来听讲座的人并不多，就算是来听讲座的人大多也是刷素拓的，但这并没有影响到她的热情。她听得很入迷，不知不觉之中似乎做出了决定，成为一名代码使就是自己未来的目标。

白露山 绫波 / 白はく露ろ山ざん　綾あや波なみ / Hakurozan Ayanami

当今古津结界外侧的最高权力者。她继承了祖先的遗志，想要恢复灵力常数。她把目光投向了古津结界。她组织了一批灵力研究者研究如何破解古津结界的限制，暗中希望能够征服另一边的世界，达成自己的愿望。

三千代 夕立

三千代 夕立 / 三さん千ぜん代だい　夕ゆう立だち / Sanzendai Yuudachi

天才的灵力研究者，时雨的妈妈。她十分向往古津结界另一边的世界，因此加入了灵力研究社。经过她长期的努力，她终于发明了可以穿过古津结界的装置。这个装置虽然一次只能转移一人，但却实现了零到一的突破，大规模的人员转移只是时间问题。而她也因此受到了绫波的赏识。

但有一天，事情发生了变化。她偶然听到绫波想要用自己发明的装置进行侵略，顿时感觉自己被利用了。此后，绫波非但没有改变自己想法的意思，还对她进行威胁。

忍无可忍的夕立终于在一天晚上带着她的女儿来到了偏僻的东风神社，用自己的装置把女儿转移了过去，随后破坏了装置。绫波闻讯赶来，并将她逮捕了起来。或许是因为她曾经的功劳，或许是希望她能够回心转意，她被免去一死，终身软禁在一个秘密的地点。

不过，以她的理想和性格，应该是不会回心转意的吧。

赤岭 胧

赤岭 胧 / 赤あか嶺みね　朧おぼろ / Akamine Oboro

在灵力研究社工作，和夕立是同事。自己的研究成果和夕立相比逊色不少，所以也不时被绫波训。不过夕立经常鼓励她，她也没有放弃。

她发明了一个可以跨越结界接收和发送电波的装置，并经常利用这个装置和结界另一边的网友对话，让她感觉到了现实世界中感受不到的氛围。

2.3：其他

其他角色

Emily Marino

Emily Marino / エミリ・マリノ / Emily Marino

古津大学的留学生，平时抱着常人没有的幻想，因此别人有时会用奇怪的目光看她

曾经在梦境中穿越古津结界，而在另一边的朋友也为自己破例穿过结界总来礼物，这让她非常触动。经过长期的资料寻找，她最终在木之叶的帮助下找到了可以稳定通过梦境穿过结界的方法。（详见《古津南柯梦》）

现在，她坚持每个月都要去结界另一边看一看自己的朋友。

西园寺 木之叶

西园寺 木之叶 / 西さい園おん寺じ　木この葉は / Saionji Konoha

和Emily比较熟悉。她原本不相信另一个世界的存在，但在Emily的帮助下找到了童年时在梦中见到的朋友，开始慢慢地相信了她。
